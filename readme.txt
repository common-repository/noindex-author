== Description ==

Simply exclude all author posts from indexing it in SERPs. This plugin allows you to noindex posts for certain authors without modifying the code and it works with most of the available themes. Just install and activate this plugin to add noindex and nofollow meta for all author posts.<br><br>Read more about this plugin on <a href="https://siriuspro.pl">WordPress</a> blog.

== Installation ==

1. Unzip the noindex-author.zip folder.
2. Upload the noindex-author folder to your /wp-content/plugins directory.
3. In your WordPress dashboard, head over to the Plugins section.
4. Activate Noindex Author plugin.
